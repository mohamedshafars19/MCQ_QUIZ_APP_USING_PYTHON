# MCQ_QUIZ_APP_USING_PYTHON

**#Description**
      This project is a simple MCQ quiz application developed using Python. The quiz contains 20 multiple-choice questions displayed sequentially. Each question provides four options, and the user selects one option as the answer. The program checks the answer using conditional statements and updates the score. After all questions are answered, the final score is displayed.
